The argument

Federal agencies publish the data they collect — flight delays, drug adverse events, hospital outcomes, water violations. The problem has never been access. It's been readability.

A CSV with 47 columns isn't usable. An AI summary of that CSV isn't the data. Open Primitive is a set of interfaces to the primary record — built so the data is something you can actually read, search, and act on.
